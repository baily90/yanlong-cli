# yanlong-cli

## setup
```
npm install yanlong-cli -g

yarn global add yanlong-cli

pnpm add -g yanlong-cli
```

## init project
```
yanlong-cli
```

## init config
```
1、输入项目名称
2、选择框架（vue/react）
3、选择包管理工具（npm/yarn/pnpm）
```

## run project
```
npm:
npm run dev

yarn:
yarn dev

pnpm:
pnpm run dev
```
